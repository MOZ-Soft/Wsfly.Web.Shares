说明：

初始化：
1:jQuery扩展初始化 
$("#wsflyCalendar").WsflyCalendar({
    ChooseDayCallback: function (options) {
        //选择日期后触发事件
    }
});
2:初始化 
var wsflyCalendarOptioins = jQuery.Wsfly.Calendar.Init({
    ContainerId:"wsflyCalendar",    //容器ID
    ChooseDayCallback: function (options) {
        //选择日期后触发事件
    }
});
属性：
名称 说明
Container 容器
ContainerId 容器ID
Target 日历对象
Inited 是否初始化
DefaultDate 默认日期 不设置时是今日
ChooseDate 选择日期
CalendarDate 日历显示月份日期
CurrentDate 当前日期（自动获取客户端日期）
AppendedStyle 是否加载样式


回调函数：
名称 参数 说明
InitedCallback options 初始化结束回调
ChooseDayCallback options 选择日期回调
ChooseMonthCallback options 月份变更回调
TodayCallback options 点击今日回调


标签设置：
名称 参数 说明
SetTagGray options,date 灰色标签
SetTagRed options,date 红色标签
SetTagGreen options,date 绿色标签
SetTagHidden options,date 隐藏标签
